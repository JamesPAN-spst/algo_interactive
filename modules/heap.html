🔵 核心定义

大根堆性质： 对所有节点 i，满足 A[parent(i)] >= A[i]（小根堆反之）。

数组存储（0-indexed）：

  • 父节点：parent(i) = (i-1) // 2
  • 左孩子：left(i) = 2*i + 1
  • 右孩子：right(i) = 2*i + 2

sift-up（上浮）伪代码：

def sift_up(heap, i):
    # 新元素插入末尾后，向上找到合适位置
    while i > 0:
        p = (i - 1) // 2          # 父节点下标
        if heap[p] < heap[i]:     # 大根堆：父 < 子，需要上浮
            heap[p], heap[i] = heap[i], heap[p]
            i = p
        else:
            break                 # 已满足堆性质，停止

sift-down（下沉）伪代码：

def sift_down(heap, i, n):
    # 堆顶被替换后，向下恢复堆性质
    while True:
        largest = i
        l, r = 2*i + 1, 2*i + 2
        if l < n and heap[l] > heap[largest]:
            largest = l
        if r < n and heap[r] > heap[largest]:
            largest = r
        if largest == i:
            break                 # 已是最大，停止
        heap[i], heap[largest] = heap[largest], heap[i]
        i = largest

🟢 典型例题详解

例题1：数据流中第K大元素（LeetCode 703）

题意： 设计一个类，在数据流中不断加入数字，每次返回当前第K大的元素。

思路： 维护大小为K的小根堆，堆顶即为第K大。新元素若大于堆顶则替换。

import heapq

class KthLargest:
    def __init__(self, k: int, nums: list):
        self.k = k
        self.heap = []                    # Python heapq 是小根堆
        for n in nums:
            self.add(n)

    def add(self, val: int) -> int:
        heapq.heappush(self.heap, val)    # 先推入
        if len(self.heap) > self.k:
            heapq.heappop(self.heap)      # 超过K个时弹出最小值
        return self.heap[0]               # 堆顶 = 第K大

时间： 每次 add O(log K)；空间： O(K)

例题2：合并K个有序链表（LeetCode 23）

题意： 合并K个已排序的链表，返回一个排序链表。

思路： 将每个链表头节点加入小根堆，每次弹出最小节点，并将其下一个节点再推入堆。

import heapq

class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def mergeKLists(lists):
    heap = []
    # 将所有链表头节点加入堆（用id避免ListNode比较报错）
    for i, node in enumerate(lists):
        if node:
            heapq.heappush(heap, (node.val, i, node))

    dummy = ListNode(0)
    cur = dummy
    while heap:
        val, i, node = heapq.heappop(heap)  # 弹出当前最小
        cur.next = node
        cur = cur.next
        if node.next:                        # 推入该链表的下一个节点
            heapq.heappush(heap, (node.next.val, i, node.next))
    return dummy.next

时间： O(N log K)，N为总节点数；空间： O(K)

例题3：查找和最小的K对数字（LeetCode 373）

题意： 给两个升序数组，找和最小的K对 (u,v)。

思路： 初始将 (nums1[i], nums2[0]) 加入小根堆，每次弹出后推入 (nums1[i], nums2[j+1])。

import heapq

def kSmallestPairs(nums1, nums2, k):
    result = []
    if not nums1 or not nums2:
        return result
    # 堆中存 (和, i, j)
    heap = [(nums1[i] + nums2[0], i, 0) for i in range(min(len(nums1), k))]
    heapq.heapify(heap)                     # O(k) 建堆

    while heap and len(result) < k:
        s, i, j = heapq.heappop(heap)
        result.append([nums1[i], nums2[j]])
        if j + 1 < len(nums2):             # 枚举 nums2 下一个
            heapq.heappush(heap, (nums1[i] + nums2[j+1], i, j+1))
    return result

时间： O(k log k)；空间： O(k)

🔴 易错点

① 数组下标从0/1引起的父子计算错误

❌ 错误（混用1-indexed）

# 0-indexed数组却用1-indexed公式
parent = i // 2        # 错！
left   = 2 * i        # 错！
right  = 2 * i + 1   # 错！

✅ 正确（0-indexed）

parent = (i - 1) // 2  # 正确
left   = 2 * i + 1    # 正确
right  = 2 * i + 2    # 正确

② 维护固定大小堆时忘记弹出

❌ 错误（只推不弹）

for n in nums:
    heapq.heappush(heap, n)
    # 忘记维护大小 → 堆无限增长
return heap[0]

✅ 正确（超K即弹）

for n in nums:
    heapq.heappush(heap, n)
    if len(heap) > k:
        heapq.heappop(heap)  # 保持大小为K
return heap[0]

③ Python实现大根堆忘记取负

❌ 错误（直接push）

# Python heapq 只有小根堆
heapq.heappush(heap, val)
# heap[0] 是最小值，不是最大值

✅ 正确（取负模拟大根堆）

heapq.heappush(heap, -val)   # 存负数
max_val = -heapq.heappop(heap)  # 取出时再取负

🔷 竞赛拓展

对顶堆维护中位数： 使用大根堆（存较小一半）+ 小根堆（存较大一半），保持两堆大小差 ≤ 1，中位数为堆顶之一或平均值。时间：每次插入 O(log n)。

import heapq

class MedianFinder:
    def __init__(self):
        self.lo = []   # 大根堆（存负数），维护较小一半
        self.hi = []   # 小根堆，维护较大一半

    def addNum(self, num):
        heapq.heappush(self.lo, -num)         # 先进大根堆
        heapq.heappush(self.hi, -heapq.heappop(self.lo))  # 平衡
        if len(self.lo) < len(self.hi):       # 保持lo不少于hi
            heapq.heappush(self.lo, -heapq.heappop(self.hi))

    def findMedian(self):
        if len(self.lo) > len(self.hi):
            return -self.lo[0]
        return (-self.lo[0] + self.hi[0]) / 2

K路归并： 将K个有序序列的首元素放入小根堆，每次弹出最小并补充该序列下一个元素，时间复杂度 O(N log K)。

Fibonacci堆： 支持 decrease-key 在 O(1) 摊销时间（普通堆为 O(log n)），使 Dijkstra 总时间降至 O(E + V log V)，但实现复杂，竞赛少用。
