⚙️ 前置知识

树的节点指针网络 散列表 (HashMap)

字典树（Trie，或者是前缀树）是专门为海量“字符串池”设计的高级武器。它的诞生就是为了回答一个 HashMap 解决不了的问题：“在这个庞大的系统字典里，有哪些词是以 `auto` 作为前缀起点的？”使用 HashMap 只能查精确匹配，而 Trie 将世界拆解为单字母的森林。

💡 推导：Trie 与 HashMap 的优劣角逐

HashMap (精确搜索 O(1))

如果要查是否存在 `apple`，Hash(apple) 是 O(L) 长度，查表 O(1)。然而，如果我要找拥有 `app` 前缀的所有单词？抱歉，哈希表只能 O(N) 全库遍历！

Trie 树 (前缀之王)

将字母打散，`apple` 就是 `a->p->p->l->e` 的树状单行道。要查以 `app` 开头的词？你只需顺着走三步，下面的整棵挂下来的庞大子树，全部都是以它为前缀的符合词汇！

Trie 的极致内存复用

如果有十万个单词：abandon, absolute, abstract...。
在 HashMap 中，每个单词的 `ab` 都是被独立申请内存存下来的字符串。十万个单词就有十万次大量的无用重复。
在 Trie 里，所有这些单词统统合并在一起走相同的 根 -> a -> b 前两次过道。这在拥有大量重叠前缀的场景极大地节省了内存！这被称为共享前缀空间。

关于终点结束符的作用

Q: 树里有 apple 和 app，如果不加区分标记，怎么辨别 app 是真实存入的词，还是仅仅是为了去 apple 的中转站？
查看答疑 这就是 isEnd (是否结尾) 这个布尔量存在的终极意义！当我们在节点 p 身上将 isEnd = true，它就声明了这不仅是过道，此时此刻截断它，这条从根到我的轨迹本身，也是一个合法的、曾经被明确插入进来的单词。

💻 模板代码：最原汁原味的字典树实现

class TrieNode:
    def __init__(self):
        # 使用哈希表储存子节点，比起一个巨大的 [None]*26 数组，能支持更多字符且节省空白空间
        self.children = {}
        # 是否是一个完整单词的结尾
        self.is_end = False

class Trie:
    def __init__(self):
        self.root = TrieNode()

    # 1. 插入 O(L) L为词长
    def insert(self, word):
        node = self.root
        for char in word:
            # 如果没有这条路，给它铺一条下去
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        # 单词全部遍历完，在这个终点节点盖个公章，确认词汇
        node.is_end = True

    # 2. 查找是否存在该完整单词 O(L)
    def search(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                return False
            node = node.children[char]
        return node.is_end  # 找到了路径不代表有词，必须看公章！

    # 3. 查找是否存在以 prefix 结尾的前缀！
    def startsWith(self, prefix):
        node = self.root
        for char in prefix:
            if char not in node.children:
                return False
            node = node.children[char]
        # 只要能顺着路走下来不死路，就意味着必然有以此为前缀的叶蔓！
        return True
操作时间空间为什么
插入/查询操作O(L)O(L)每次插入完全忽略词典大小 $N$，永远只和这个词自身的长度 $L$ 有关！
极端恶劣输入-O(N $ imes$ L)如果完全没有任意两个词共用任何前缀，全部退化成平摊。
