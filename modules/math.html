初等数论 位运算基础

算法中的数学模块不同于高数，它侧重离散数学和数论。核心要求是：在保证不溢出的前提下，将复杂的数学定义转化为计算机可高效执行的（位运算、模运算）程序逻辑。

💡 推导：核心数学定理与公式

1. 欧几里得算法 (Euclidean Algorithm - 求最大公约数 GCD)

核心思想： `gcd(a, b) = gcd(b, a % b)`。大数除以小数的余数，一定会继承它们共有的因子。一旦较小数缩小到 0，剩下的那个大数就是最大公因数。
Q: 为什么 gcd(a,b) = gcd(b, a%b)？
查看答疑 假设我们要求 gcd(A, B)且 A > B。A 可以写成 A = k*B + R，其中 R 就是 A % B。如果有一个数能同时整除 A 和 B，那么它必然也能整除 R = A - k*B。因此，A 和 B 的公约数集合，完全等价于 B 和 R 的公约数集合。不断替换成更小的数字组合，直到 0 出现。

2. 快速幂 (Fast Exponentiation)

线性连乘 (Naive)

计算 `x^n`，从 1 循环乘到 n，时间复杂度 O(N)。如果 N=$10^9$ 会超时。

二进制拆分 (Smart)

因为 $13 = 1101_2$，所以 $x^{13} = x^8 * x^4 * x^1$。我们只需要不断把自身的底数平方，并在看到幂次的二进制位是 1 时乘入答案。O(log N)。

3. 埃拉托斯特尼筛法 (Sieve of Eratosthenes - 找素数)

1
策略： 从 2 开始，如果它是素数，就把它的所有倍数（4,6,8..）涂黑标记为合数。
2
重要优化： 如果当前素数是 `p`，没必要从 `p*2` 开始涂黑，直接从 `p*p` 开始涂黑即可！因为比它小的倍数 (如 `p*2`, `p*3`...) 肯定早在处理 2 和 3 时就被涂黑过了。

4. 模运算与费马小定理 -> 乘法逆元

加减乘可以直接对中间结果取模：(a * b) % p == ((a % p) * (b % p)) % p。
但是除法不行！ (a / b) % p != (a % p) / (b % p)。必须求 b 在模 p 下的逆元 b^-1。
费马小定理指出，如果 p 是质数，那么 $b^{p-1} \equiv 1 \pmod p$。两边同除以 b 得 $b^{p-2} \equiv b^{-1} \pmod p$。因此，除以 b，就等于乘以 $b^{p-2}$ 并取模！

💻 模板代码：GCD、快速幂与素数筛

# 1. 欧几里得求 GCD 与 LCM
def gcd(a, b):
    while b != 0:
        a, b = b, a % b
    return a

def lcm(a, b):
    # 先除后乘防止溢出 (Python其实无所谓，C++很重要)
    return a // gcd(a, b) * b

# 2. 快速幂 (带取模)
def fastPow(x, n, mod):
    res = 1
    x = x % mod
    while n > 0:
        if n & 1:           # 如果当前最低位为 1
            res = (res * x) % mod
        x = (x * x) % mod   # 底数不断翻脸平方
        n >>= 1             # 右移一位
    return res

# 3. 乘法逆元 (如果 mod 是质数)
def modInverse(b, mod):
    return fastPow(b, mod - 2, mod)

# 4. 埃氏筛找 n 以内的素数
def sieve(n):
    is_prime = [True] * (n + 1)
    is_prime[0] = is_prime[1] = False
    for i in range(2, int(n**0.5) + 1):
        if is_prime[i]:
            # 从 i^2 开始涂黑
            for j in range(i * i, n + 1, i):
                is_prime[j] = False
    return [i for i, p in enumerate(is_prime) if p]

🔴 易错点分析

❌ 错误的逆元使用前提

# 需要计算 (a / b) % m
ans = (a % m) / (b % m)
# 或者随便用费马小定理，哪怕 m 不是质数：
ans = a * fastPow(b, m-2, m) % m

✅ 费马小必须保证 m 是质数

# 如果 m 是质数 (如 10^9+7)：
ans = (a * modInverse(b, m)) % m

# 如果不是质数，必须使用 扩展欧几里得(exgcd) 求解

🔷 竞赛拓展

欧拉筛法 (线性筛)： 埃氏筛法由于 12 会被 2 和 3 重复涂黑，仍有一些浪费。线性筛法保证每个合数只被其“最小素因子”筛掉一次，绝对的 O(N) 性能，常用于带有很多求积性函数的情境。

组合数学 (杨辉三角与预处理乘阶)： 在计算 $C(n,k) \pmod p$ 时，一般预处理所有数字的阶乘 `fact[i]` 和阶乘逆元 `inv[i]`，然后直接查表 O(1) 返回 `fact[n]*inv[k]*inv[n-k] % p`。
