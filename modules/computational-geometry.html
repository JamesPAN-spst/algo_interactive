向量运算基础 坐标系概念

计算几何主要处理平面内的点、线。将点表示为向量（Vector）可以避免繁琐的解析几何方程，用叉积优雅地解决方向问题。

💡 推导：叉积如何指示转弯方向？

核心思想： 向量叉积的符号等价于右手系的 Z 轴。正值表示逆时针（左转），负值表示顺时针（右转）。

解析几何 (Naive)

求斜率 k，比较 y。遇垂直线需特判且有除法精度问题。

向量叉积 (Smart)

把线段视作向量，仅用乘减 `x1*y2 - x2*y1`。无除法，无 ∞ 特判。

1
向量坐标化： 设原点 $O(0,0)$，$A(x_1, y_1)$，$B(x_2, y_2)$。向量 $ ec{OA} = (x_1, y_1)$，$ ec{OB} = (x_2, y_2)$。
2
计算与判定： $cross = x_1 y_2 - x_2 y_1$。为正时 B 在 OA 逆时针侧，为负时在顺时针侧，0 时共线。
Q: 为什么叉积的符号就能给出方向？
查看答疑 2D 中的向量叉积可视为 3D 向量 (x,y,0) × (x',y',0)。其结果只在 Z 轴有值，服从右手法则：大拇指朝上（正）即逆时针，朝下（负）即顺时针。
Q: Graham 扫描法如何确保最后是凸包？
查看答疑 按极角排序后，我们按逆时针顺序绕点集一圈。只要我们每一步都是“向左转” (叉积>0)，最终必然画出一个凸多边形。如果有“右转”，就说明之前的点凹进去了，退栈即可。

💻 模板代码与复杂度

# 1. 基础向量操作
EPS = 1e-9

class Point:
    def __init__(self, x, y):
        self.x, self.y = x, y
    def __sub__(self, other):
        return Point(self.x - other.x, self.y - other.y)

def cross(O, A, B):
    # 返回 >0 说明 B 在 OA 逆时针侧
    return (A.x - O.x) * (B.y - O.y) - (A.y - O.y) * (B.x - O.x)

# 2. Andrew 凸包 (Graham扫面变种)
def convex_hull(points):
    points = sorted(set(points)) # x 排序
    if len(points) <= 1: return points

    lower = []
    for p in points:
        while len(lower) >= 2 and cross(lower[-2], lower[-1], p) <= 0:
            lower.pop() # 右转或共线则弹出
        lower.append(p)

    upper = []
    for p in reversed(points):
        while len(upper) >= 2 and cross(upper[-2], upper[-1], p) <= 0:
            upper.pop()
        upper.append(p)

    return lower[:-1] + upper[:-1]
技巧时间空间为什么
叉积判断O(1)O(1)仅乘减运算。
凸包(Andrew)O(n log n)O(n)时间由初始的点的排序主导，后续遍历操作均为摊还 O(1)。

🟢 典型例题：凸包变形

变种：如果我们要边界上的所有共线点怎么办？

标准的凸包算法中 cross <= 0 时退栈以剔除共线中间点。如果需要包含边界上所有点，则需要改为 cross < 0，同时最后不要弹出共线点。

🔴 易错点分析

❌ 错误判断共线

if cross(O, A, B) == 0:
    print("共线")

✅ 正确判断共线 (加 EPS)

EPS = 1e-9
if abs(cross(O, A, B)) < EPS:
    print("共线")

旋转卡壳（Rotating Calipers）： 线性扫描凸包用于求最远两点（直径）。通过维护两根平行的扫描线在多边形外部滚动，O(n) 内可以求解。

如何判断点在凸包内： 相比于射线法的 O(n)，如果点集是凸包，可以二分查找，达到 O(log n)。
