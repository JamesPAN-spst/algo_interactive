🔵 核心定义

基本位运算符与常用技巧汇总：

# ===== 基本操作 =====
x & y    # 按位与：同为1才为1
x | y    # 按位或：有1就为1
x ^ y    # 按位异或：相同为0，不同为1
~x       # 按位取反（Python中 ~x = -x-1）
x << n   # 左移n位（×2^n）
x >> n   # 右移n位（÷2^n，Python无符号）

# ===== 常用技巧 =====
x & (x-1)          # 消去最低位的1（判断2的幂：x>0 and not x&(x-1)）
x & (-x)           # lowbit：取最低位的1（树状数组核心）
x ^ x = 0         # 自身异或为0（去重神器）
x ^ 0 = x         # 与0异或不变
(x >> i) & 1       # 取第i位（0-indexed）
x | (1 << i)       # 将第i位置1
x & ~(1 << i)      # 将第i位清0
x ^ (1 << i)       # 翻转第i位

# ===== 子集枚举 =====
n = 4
for mask in range(1 << n):        # 枚举n位的所有子集
    # mask是0到2^n-1的每个状态
    bits = bin(mask).count('1')   # 统计1的个数
    for i in range(n):
        if mask >> i & 1:         # 第i位是否选中
            pass

# 枚举mask的所有子集（subset of subset）
sub = mask
while sub:
    # sub是mask的一个子集
    sub = (sub - 1) & mask        # Brian Kernighan技巧获取下一个子集

📊 复杂度速查

操作时间复杂度说明
基本位运算（&, |, ^, ~, <<, >>）O(1)单次操作
统计1的个数（bin().count）O(log n)Python内置
枚举n位所有子集O(2ⁿ)共2ⁿ个状态
枚举mask的所有子集O(2^|mask|)对所有mask总计3^n
状压DP（n个状态）O(2ⁿ × n)n≤20时可接受

🟢 典型例题详解

例题1：只出现一次的数字（LeetCode 136）— 异或

题意： 除一个数外其余都出现两次，找出只出现一次的数。

def singleNumber(nums):
    result = 0
    for num in nums:
        result ^= num    # 两次异或抵消为0，剩下的就是只出现一次的
    return result
    # 时间O(n)，空间O(1) — 比哈希表方法节省空间
例题2：两整数之和（不用+/-，LeetCode 371）— 位运算模拟加法

题意： 不使用+/-，用位运算计算两整数之和。

def getSum(a, b):
    # Python整数无固定位宽，需要掩码处理
    MASK = 0xFFFFFFFF
    MAX  = 0x7FFFFFFF
    while b != 0:
        carry = (a & b) << 1    # 进位 = 两数都为1的位，左移一位
        a = (a ^ b) & MASK      # 无进位的和（XOR），截断32位
        b = carry & MASK         # 新的进位
    # 处理负数（32位补码转Python负数）
    return a if a <= MAX else ~(a ^ MASK)

时间： O(1)（最多32次循环）；空间： O(1)

例题3：数组中数字出现的次数（两个只出现一次的数）

题意： 数组中两个数只出现一次，其余都出现两次，找出这两个数。

def singleNumbers(nums):
    xor = 0
    for n in nums: xor ^= n    # a ^ b（两个单数的异或）

    # 找到xor中任意一个为1的位（a和b在该位不同）
    mask = xor & (-xor)        # lowbit：取最低位的1

    a = b = 0
    for n in nums:
        if n & mask:            # 根据该位将数组分成两组
            a ^= n             # 每组内相同数消除，剩下a或b
        else:
            b ^= n
    return [a, b]

时间： O(n)；空间： O(1)

🔴 易错点

① 位运算优先级低，忘加括号

❌ 错误（& 比 == 优先级低）

if x & 1 == 0:    # 等价于 x & (1==0) = x&False = x&0
    pass          # 条件永远为False！

✅ 正确（加括号）

if (x & 1) == 0:  # 先算x&1，再与0比较
    pass
# 或更简洁：
if not x & 1:     # x&1==0 即x为偶数
    pass

② Python ~取反与C不同（结果为负数）

❌ 误解（~x不是无符号取反）

x = 5             # 0b00000101
~x                # Python: -6（无限精度补码）
                  # 不是0b11111010

✅ 正确（用掩码模拟32位）

MASK = 0xFFFFFFFF
result = (~x) & MASK   # 模拟32位无符号取反
# 或：x ^ MASK         # 同等效果

🔷 竞赛拓展

按位DP： 枚举每一位的贡献，O(n log max_val) 计算所有对的AND/OR/XOR总和。

def totalHammingDistance(nums):
    # 所有对的汉明距离之和（各位独立计算）
    total = 0
    for bit in range(32):
        ones = sum((n >> bit) & 1 for n in nums)  # 该位为1的个数
        zeros = len(nums) - ones
        total += ones * zeros   # 一个1与一个0构成一对汉明距离
    return total

布隆过滤器（Bloom Filter）： 多个哈希函数 + 位数组，O(k)查询，有一定假阳性率但无假阴性。适合大数据集合的快速存在性判断。

Gray码： 相邻数字只有一位不同，n位Gray码 = n XOR (n>>1)。用于减少状态切换次数。
