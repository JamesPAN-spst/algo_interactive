🔵 核心定义

向量与叉积（核心工具）：

EPS = 1e-9    # 浮点误差阈值

class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    def __sub__(self, other):
        return Point(self.x - other.x, self.y - other.y)
    def __add__(self, other):
        return Point(self.x + other.x, self.y + other.y)

def cross(O, A, B):
    """向量OA × OB 的z分量（叉积）
       > 0: B在OA左侧（逆时针）
       < 0: B在OA右侧（顺时针）
       = 0: 三点共线"""
    return (A.x - O.x) * (B.y - O.y) - (A.y - O.y) * (B.x - O.x)

def dot(O, A, B):
    """向量OA · OB 的点积（用于角度/投影）"""
    return (A.x - O.x) * (B.x - O.x) + (A.y - O.y) * (B.y - O.y)

def dist(A, B):
    """两点距离"""
    return ((A.x-B.x)**2 + (A.y-B.y)**2) ** 0.5

def on_segment(P, A, B):
    """判断点P是否在线段AB上"""
    return (abs(cross(P, A, B)) < EPS and
            min(A.x, B.x) - EPS <= P.x <= max(A.x, B.x) + EPS and
            min(A.y, B.y) - EPS <= P.y <= max(A.y, B.y) + EPS)

线段相交判断：

def segments_intersect(A, B, C, D):
    """判断线段AB与CD是否相交"""
    d1 = cross(C, D, A)    # A相对于CD的方向
    d2 = cross(C, D, B)    # B相对于CD的方向
    d3 = cross(A, B, C)    # C相对于AB的方向
    d4 = cross(A, B, D)    # D相对于AB的方向

    if d1 * d2 < 0 and d3 * d4 < 0:
        return True         # 严格相交（各在对方两侧）

    # 端点在另一线段上的特殊情况
    if abs(d1) < EPS and on_segment(A, C, D): return True
    if abs(d2) < EPS and on_segment(B, C, D): return True
    if abs(d3) < EPS and on_segment(C, A, B): return True
    if abs(d4) < EPS and on_segment(D, A, B): return True
    return False

🖼 结构图示：叉积的几何含义

叉积 cross(O, A, B) = (A-O) × (B-O):
         B         cross > 0: B在OA左侧（逆时针）
        /
   O--→A           cross < 0: B在OA右侧（顺时针）
   
         A         cross = 0: O、A、B共线
        /
   O--→B

|cross(O,A,B)| = 平行四边形OA-OB的面积
              = 三角形OAB面积×2

多边形面积 Shoelace公式：
  Area = |Σ(x_i * y_{i+1} - x_{i+1} * y_i)| / 2
  按顶点顺序累加叉积之和的一半

凸包Graham扫描:
  1. 找最低最左点作为基点P
  2. 其他点按与P的极角排序
  3. 维护单调栈：新点使栈顶的右转则弹出
  栈中剩余点即为凸包顶点

🟢 典型例题详解

例题1：凸包（Andrew单调链）

题意： 给一组点，求凸包（包含所有点的最小凸多边形）的顶点。

def convex_hull(points):
    points = sorted(set(points))        # 按x升序（x相同则y升序）
    n = len(points)
    if n <= 1: return points

    # 构建下凸壳（从左到右）
    lower = []
    for p in points:
        while (len(lower) >= 2 and
               cross(lower[-2], lower[-1], p) <= 0):  # 右转或共线则弹出
            lower.pop()
        lower.append(p)

    # 构建上凸壳（从右到左）
    upper = []
    for p in reversed(points):
        while (len(upper) >= 2 and
               cross(upper[-2], upper[-1], p) <= 0):
            upper.pop()
        upper.append(p)

    # 去掉两端重复点，合并
    return lower[:-1] + upper[:-1]     # 凸包顶点（逆时针）

时间： O(n log n)；空间： O(n)

例题2：多边形面积（Shoelace公式） 
def polygon_area(points):
    """计算多边形面积（顶点按顺序排列）"""
    n = len(points)
    area = 0
    for i in range(n):
        j = (i + 1) % n
        area += points[i][0] * points[j][1]
        area -= points[j][0] * points[i][1]
    return abs(area) / 2

# 例：三角形 [(0,0),(4,0),(0,3)]
# area = |0*0-4*0 + 4*3-0*0 + 0*0-0*3| / 2 = |12| / 2 = 6 ✓
例题3：最近点对（分治法）

题意： 平面上n个点，找最近两点的距离。

import math

def closest_pair(points):
    def dist(p1, p2):
        return math.sqrt((p1[0]-p2[0])**2 + (p1[1]-p2[1])**2)

    def rec(ps):
        n = len(ps)
        if n <= 3:
            # 暴力求小范围内最近点对
            min_d = float('inf')
            for i in range(n):
                for j in range(i+1, n):
                    min_d = min(min_d, dist(ps[i], ps[j]))
            return min_d

        mid = n // 2
        mid_x = ps[mid][0]
        d = min(rec(ps[:mid]), rec(ps[mid:]))   # 左右各自最近距离

        # 合并：只需考虑中线两侧宽度d内的点
        strip = [p for p in ps if abs(p[0] - mid_x) < d]
        strip.sort(key=lambda p: p[1])           # 按y排序

        for i in range(len(strip)):
            j = i + 1
            while j < len(strip) and strip[j][1] - strip[i][1] < d:
                d = min(d, dist(strip[i], strip[j]))
                j += 1
        return d

    points.sort()  # 按x坐标排序
    return rec(points)

时间： O(n log n)；空间： O(n)

🔴 易错点

① 浮点数直接用==比较导致误判

❌ 错误（直接比较浮点）

if cross(O, A, B) == 0:    # 浮点误差可能使共线点不等于0
    print("共线")           # 1e-15这样的误差会导致漏判

✅ 正确（加EPS误差阈值）

EPS = 1e-9
if abs(cross(O, A, B)) < EPS:   # 误差范围内视为0
    print("共线")

② 叉积符号方向理解错误

❌ 错误（混淆左右）

if cross(O, A, B) > 0:
    print("B在OA右侧")   # 错！>0表示逆时针，即B在OA左侧

✅ 正确

if cross(O, A, B) > 0:
    print("B在OA左侧（逆时针）")   # ✓
if cross(O, A, B) < 0:
    print("B在OA右侧（顺时针）")   # ✓

🔷 竞赛拓展

扫描线算法： 用一条竖线从左到右扫过平面，维护活跃事件集合（用BST）。用于：矩形面积并（O(n log n)）、线段交点统计（Shamos-Hoey O(n log n)）。

半平面交： 每个不等式约束对应一个半平面，求多个半平面的交集（凸多边形）。O(n log n)，应用于线性规划可行域可视化。

旋转卡壳（Rotating Calipers）： 在凸包上用两个对径点扫描，O(n)内求直径、最宽两边对、最近点对等。

def convex_hull_diameter(hull):
    """凸包直径（旋转卡壳）"""
    n = len(hull)
    if n == 1: return 0
    if n == 2: return dist(hull[0], hull[1])
    diameter_sq = 0
    j = 1
    for i in range(n):
        next_i = (i + 1) % n
        while (cross(hull[i], hull[next_i], hull[(j+1)%n]) >
               cross(hull[i], hull[next_i], hull[j])):
            j = (j + 1) % n
        diameter_sq = max(diameter_sq,
                          dist(hull[i], hull[j])**2,
                          dist(hull[next_i], hull[j])**2)
    return diameter_sq ** 0.5
