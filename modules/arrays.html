🔵 核心定义

左闭右开区间： [left, right)，即 left 包含，right 不包含。初始化 left=0, right=n，循环条件 left < right。

前缀和定义：

pre[0] = 0
pre[i] = pre[i-1] + nums[i-1]   # pre[i] 表示前i个元素之和
# 区间和 [l, r)（0-indexed）: sum = pre[r] - pre[l]

滑动窗口模板：

def sliding_window(s, k):
    left = 0
    window = {}                       # 记录窗口内状态
    result = 0
    for right in range(len(s)):
        # 1. 扩展窗口，加入 s[right]
        window[s[right]] = window.get(s[right], 0) + 1
        # 2. 当窗口不满足条件时收缩
        while 不满足条件:
            window[s[left]] -= 1
            if window[s[left]] == 0:
                del window[s[left]]
            left += 1
        # 3. 更新答案
        result = max(result, right - left + 1)
    return result

🟢 典型例题详解

例题1：移动零（LeetCode 283）— 同向双指针

题意： 将数组中所有0移到末尾，保持非0元素相对顺序，原地操作。

思路： slow 指向下一个非0元素该放的位置，fast 扫描所有非0元素。

def moveZeroes(nums):
    slow = 0                         # 慢指针：非0元素填充位
    for fast in range(len(nums)):    # 快指针：扫描整个数组
        if nums[fast] != 0:
            nums[slow] = nums[fast]  # 将非0元素前移
            slow += 1
    # slow之后全部填0
    for i in range(slow, len(nums)):
        nums[i] = 0
    # 时间O(n)，空间O(1)

时间： O(n)；空间： O(1)

例题2：无重复字符的最长子串（LeetCode 3）— 滑动窗口

题意： 找字符串中无重复字符的最长子串长度。

思路： 哈希表记录每个字符的最新位置，right扩展，发现重复时left跳过。

def lengthOfLongestSubstring(s):
    char_pos = {}                    # 记录字符最新出现位置
    left = 0
    ans = 0
    for right, ch in enumerate(s):
        if ch in char_pos and char_pos[ch] >= left:
            left = char_pos[ch] + 1  # left跳过重复字符
        char_pos[ch] = right         # 更新最新位置
        ans = max(ans, right - left + 1)
    return ans

时间： O(n)；空间： O(字符集大小)

例题3：子数组和为K（LeetCode 560）— 前缀和+哈希

题意： 找和为k的子数组的个数。

思路： 前缀和 pre[j]-pre[i]=k，即 pre[i]=pre[j]-k，哈希表统计前缀和出现次数。

def subarraySum(nums, k):
    count = 0
    pre_sum = 0
    freq = {0: 1}                    # pre[0]=0 出现1次
    for num in nums:
        pre_sum += num               # 累加前缀和
        # 找有多少个之前的前缀和 = pre_sum - k
        count += freq.get(pre_sum - k, 0)
        freq[pre_sum] = freq.get(pre_sum, 0) + 1
    return count

时间： O(n)；空间： O(n)

🔴 易错点

① 窗口收缩条件写反导致死循环

❌ 错误（条件方向反）

while len(window) <= k:  # 条件反了
    # 永远不会收缩，死循环
    left += 1

✅ 正确

while len(window) > k:   # 超出k时才收缩
    window[s[left]] -= 1
    left += 1

② 前缀和下标偏移混淆

❌ 错误（下标不统一）

pre = [0] * n            # 长度n，缺少pre[0]
for i in range(n):
    pre[i] = pre[i-1] + nums[i]
# pre[-1] 在Python里是最后元素，引发逻辑错误

✅ 正确（长度n+1）

pre = [0] * (n + 1)      # 多一位存pre[0]=0
for i in range(1, n+1):
    pre[i] = pre[i-1] + nums[i-1]
# 区间[l,r]之和 = pre[r+1] - pre[l]

③ 对撞双指针移动时机错误

❌ 错误（两个都移动）

if nums[l] + nums[r] < target:
    l += 1
    r -= 1   # 错误！不该同时移动

✅ 正确（只移一个）

if nums[l] + nums[r] < target:
    l += 1   # 和小了，左指针右移
elif nums[l] + nums[r] > target:
    r -= 1   # 和大了，右指针左移

🔷 竞赛拓展

单调队列解决滑窗最值： 用双端队列维护窗口内单调递减序列，O(n) 求固定窗口大小k的最大值。

from collections import deque

def maxSlidingWindow(nums, k):
    dq = deque()           # 存下标，队首始终是当前窗口最大值下标
    result = []
    for i, num in enumerate(nums):
        # 移除超出窗口的下标
        while dq and dq[0] < i - k + 1:
            dq.popleft()
        # 维护单调递减：移除所有比当前小的下标
        while dq and nums[dq[-1]] < num:
            dq.pop()
        dq.append(i)
        if i >= k - 1:
            result.append(nums[dq[0]])   # 窗口最大值
    return result

差分数组区间更新： diff[l] += val; diff[r+1] -= val，最后前缀和还原，O(1) 批量区间操作。

离散化： 数据范围大但个数少时，将值映射到连续下标，配合前缀和或线段树使用。
