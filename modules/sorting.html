🔵 核心定义

快速排序（Lomuto 分区）：

def quicksort(arr, lo, hi):
    if lo >= hi: return
    pivot = arr[hi]          # 选最后一个元素为pivot
    i = lo - 1               # i指向小于pivot区域的末尾
    for j in range(lo, hi):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    arr[i+1], arr[hi] = arr[hi], arr[i+1]  # pivot就位
    p = i + 1
    quicksort(arr, lo, p - 1)
    quicksort(arr, p + 1, hi)

归并排序：

def mergesort(arr, lo, hi):
    if lo >= hi: return
    mid = (lo + hi) // 2
    mergesort(arr, lo, mid)
    mergesort(arr, mid+1, hi)
    # 合并两个有序子数组
    left = arr[lo:mid+1]            # 拷贝左半部分
    right = arr[mid+1:hi+1]         # 拷贝右半部分
    i = j = 0
    k = lo
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:     # 保持稳定性：等号取左边
            arr[k] = left[i]; i += 1
        else:
            arr[k] = right[j]; j += 1
        k += 1
    while i < len(left): arr[k] = left[i]; i += 1; k += 1
    while j < len(right): arr[k] = right[j]; j += 1; k += 1

🟢 典型例题详解

例题1：数组中的第K个最大元素（LeetCode 215）— 快速选择

题意： 找数组中第K大元素，要求时间优于O(n log n)。

思路： 快速选择（Quickselect）：分区后只递归一侧，平均O(n)。

import random

def findKthLargest(nums, k):
    # 转化：第k大 = 排序后第 n-k 个（0-indexed）
    target = len(nums) - k

    def quickselect(lo, hi):
        pivot_idx = random.randint(lo, hi)   # 随机化避免最坏情况
        nums[pivot_idx], nums[hi] = nums[hi], nums[pivot_idx]
        pivot = nums[hi]
        i = lo - 1
        for j in range(lo, hi):
            if nums[j] <= pivot:
                i += 1
                nums[i], nums[j] = nums[j], nums[i]
        nums[i+1], nums[hi] = nums[hi], nums[i+1]
        p = i + 1
        if p == target: return nums[p]       # 找到了
        elif p < target: return quickselect(p+1, hi)
        else: return quickselect(lo, p-1)

    return quickselect(0, len(nums)-1)

时间： 平均O(n)，最坏O(n²)；空间： O(log n)

例题2：合并区间（LeetCode 56）— 排序预处理

题意： 给一组区间，合并所有重叠区间。

def merge(intervals):
    intervals.sort(key=lambda x: x[0])   # 按起点排序
    result = [intervals[0]]

    for start, end in intervals[1:]:
        if start <= result[-1][1]:        # 与上一个区间重叠
            result[-1][1] = max(result[-1][1], end)  # 扩展右端点
        else:
            result.append([start, end])  # 不重叠，新增区间
    return result

时间： O(n log n)（排序主导）；空间： O(n)

例题3：计数排序（非比较）— 颜色分类（LeetCode 75）

题意： 将只含0、1、2的数组排序（荷兰国旗问题）。

def sortColors(nums):
    lo, mid, hi = 0, 0, len(nums) - 1
    while mid <= hi:
        if nums[mid] == 0:
            nums[lo], nums[mid] = nums[mid], nums[lo]
            lo += 1; mid += 1          # 0和lo交换，两指针前进
        elif nums[mid] == 1:
            mid += 1                   # 1直接跳过
        else:
            nums[mid], nums[hi] = nums[hi], nums[mid]
            hi -= 1                    # 2和hi交换，hi后退（mid不动，需检查换来的）

时间： O(n)；空间： O(1) — 一次遍历，三路分区

🔴 易错点

① 快排分区边界处理错误

❌ 错误（pivot放中间但边界错）

def partition(arr, lo, hi):
    pivot = arr[lo]
    i, j = lo, hi
    while i < j:
        while arr[j] >= pivot: j -= 1
        while arr[i] <= pivot: i += 1  # 漏了 i

✅ 正确（加i<j保护）

def partition(arr, lo, hi):
    pivot = arr[lo]
    i, j = lo, hi
    while i < j:
        while i < j and arr[j] >= pivot: j -= 1
        while i < j and arr[i] <= pivot: i += 1
        if i < j:
            arr[i], arr[j] = arr[j], arr[i]
    arr[lo], arr[i] = arr[i], arr[lo]
    return i

② 归并时辅助数组拷贝顺序错误

❌ 错误（直接原地修改）

# 合并时直接操作原数组
# 右指针移动会覆盖左半部分数据
arr[k] = arr[i]  # arr[i]可能已被覆盖

✅ 正确（先拷贝再合并）

tmp = arr[lo:hi+1]  # 先拷贝整段
i, j = 0, mid - lo + 1
for k in range(lo, hi+1):
    # 从tmp中读取，安全
    if i > mid-lo: arr[k] = tmp[j]; j+=1
    elif j > hi-lo: arr[k] = tmp[i]; i+=1
    elif tmp[i] <= tmp[j]: arr[k] = tmp[i]; i+=1
    else: arr[k] = tmp[j]; j+=1

🔷 竞赛拓展

外部排序： 数据量超出内存时，分块排序后多路归并，磁盘I/O为主要瓶颈。

Timsort： Python内置排序算法，归并+插入排序混合。对部分有序数据接近O(n)，最坏O(n log n)。

逆序对计数： 归并排序可在O(n log n)内统计逆序对，合并时右侧元素先被选中则左剩余元素都与其构成逆序对。

def countInversions(arr):
    count = 0
    def mergesort(arr):
        nonlocal count
        if len(arr) <= 1: return arr
        mid = len(arr) // 2
        left = mergesort(arr[:mid])
        right = mergesort(arr[mid:])
        merged = []
        i = j = 0
        while i < len(left) and j < len(right):
            if left[i] <= right[j]:
                merged.append(left[i]); i += 1
            else:
                count += len(left) - i    # 左边剩余元素都比right[j]大
                merged.append(right[j]); j += 1
        return merged + left[i:] + right[j:]
    mergesort(arr)
    return count
