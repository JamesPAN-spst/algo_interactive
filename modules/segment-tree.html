🔵 核心定义

线段树（区间和）完整实现：

class SegmentTree:
    def __init__(self, nums):
        n = len(nums)
        self.n = n
        self.tree = [0] * (4 * n)     # 4n保证空间够用
        self.lazy = [0] * (4 * n)     # 懒标记（延迟批量更新）
        self._build(nums, 1, 0, n-1)  # 从节点1开始（1-indexed更方便）

    def _build(self, nums, node, start, end):
        if start == end:
            self.tree[node] = nums[start]
            return
        mid = (start + end) // 2
        self._build(nums, 2*node,   start,   mid)
        self._build(nums, 2*node+1, mid+1,   end)
        self.tree[node] = self.tree[2*node] + self.tree[2*node+1]

    def _pushdown(self, node, start, end):
        """将懒标记下传给子节点"""
        if self.lazy[node]:
            mid = (start + end) // 2
            lc, rc = 2*node, 2*node+1
            self.tree[lc] += self.lazy[node] * (mid - start + 1)
            self.tree[rc] += self.lazy[node] * (end - mid)
            self.lazy[lc] += self.lazy[node]
            self.lazy[rc] += self.lazy[node]
            self.lazy[node] = 0         # 清除当前懒标记

    def update_range(self, node, start, end, l, r, val):
        """区间[l,r]每个元素加val"""
        if r < start or end < l: return     # 无交集
        if l <= start and end <= r:          # 被完全覆盖
            self.tree[node] += val * (end - start + 1)
            self.lazy[node] += val
            return
        self._pushdown(node, start, end)    # 先下传
        mid = (start + end) // 2
        self.update_range(2*node,   start,   mid, l, r, val)
        self.update_range(2*node+1, mid+1,   end, l, r, val)
        self.tree[node] = self.tree[2*node] + self.tree[2*node+1]

    def query(self, node, start, end, l, r):
        """查询[l,r]区间和"""
        if r < start or end < l: return 0
        if l <= start and end <= r: return self.tree[node]
        self._pushdown(node, start, end)
        mid = (start + end) // 2
        return (self.query(2*node,   start, mid, l, r) +
                self.query(2*node+1, mid+1, end, l, r))

树状数组（BIT）对比——仅支持前缀和：
BIT[i] = 原数组中 [i-lowbit(i)+1, i] 的和
lowbit(i) = i & (-i)
update: 从i向上跳 i += lowbit(i)
query:  从i向下跳 i -= lowbit(i)

🟢 典型例题详解

例题1：区间求和（LeetCode 307）— 树状数组BIT

题意： 实现数组的单点更新和区间求和，操作次数多。

class BIT:                              # Binary Indexed Tree (树状数组)
    def __init__(self, nums):
        n = len(nums)
        self.n = n
        self.bit = [0] * (n + 1)       # 1-indexed
        for i, v in enumerate(nums):
            self._update(i + 1, v)

    def _update(self, i, delta):
        while i <= self.n:
            self.bit[i] += delta
            i += i & (-i)              # 加上lowbit，向上跳

    def _prefix(self, i):              # 前缀和 [1..i]
        s = 0
        while i > 0:
            s += self.bit[i]
            i -= i & (-i)              # 减去lowbit，向下跳
        return s

    def update(self, i, val):
        old = self.nums[i]
        self.nums[i] = val
        self._update(i + 1, val - old)

    def sumRange(self, left, right):
        return self._prefix(right+1) - self._prefix(left)

时间： O(log n) 每次操作；空间： O(n)

例题2：航班预订统计（LeetCode 1109）— 差分数组

题意： n个航班，多次订座操作[first,last,seats]，求每个航班实际预订量。

def corpFlightBookings(bookings, n):
    diff = [0] * (n + 2)               # 差分数组（多一位防越界）
    for first, last, seats in bookings:
        diff[first] += seats           # 起点加
        diff[last+1] -= seats          # 终点后一位减
    # 还原：前缀和即为实际值
    result = []
    cur = 0
    for i in range(1, n+1):
        cur += diff[i]
        result.append(cur)
    return result

时间： O(n + m)；空间： O(n)

🔴 易错点

① pushdown时机错误（区间更新前未下传）

❌ 错误（更新前没pushdown）

def update(self, node, start, end, l, r, val):
    if r < start or end < l: return
    if l <= start <= end <= r:
        self.tree[node] += val*(end-start+1)
        self.lazy[node] += val
        return
    # 忘记pushdown！子节点的lazy没有下传
    mid = (start+end)//2
    self.update(2*node, start, mid, l, r, val)
    self.update(2*node+1, mid+1, end, l, r, val)

✅ 正确（先pushdown再递归）

def update(self, node, start, end, l, r, val):
    if r < start or end < l: return
    if l <= start <= end <= r:
        self.tree[node] += val*(end-start+1)
        self.lazy[node] += val
        return
    self._pushdown(node, start, end)  # 先下传！
    mid = (start+end)//2
    self.update(...)
    self.tree[node] = self.tree[2*node] + self.tree[2*node+1]

② 树状数组update时delta计算错误

❌ 错误（直接用新值）

def update(self, i, val):
    self._update(i+1, val)     # 错！应传差值delta
    # BIT存的是增量，直接传val会累加而非替换

✅ 正确（传差值）

def update(self, i, val):
    delta = val - self.nums[i]     # 新值 - 旧值
    self.nums[i] = val             # 更新原始数组
    self._update(i+1, delta)       # BIT加上差值

🔷 竞赛拓展

可持久化线段树（主席树）： 每次操作创建新版本，共享未修改节点，O(log n)空间/时间，支持区间第K小查询（历史版本）。

线段树合并： 将两棵动态开点线段树在O(M log N)时间合并，用于树上DP降维（如统计子树内排名等）。

Li Chao线段树： 维护线集合，O(log V)查询某x处最小/大y值（插入直线，查询静态位置的极值）。用于斜率优化DP的维护。
