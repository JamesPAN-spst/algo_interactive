知识点总结提纲（Knowledge / Connaissances）

🎯 引子：为什么 n & (n-1) 能判断 2 的幂？

2 的幂（1, 2, 4, 8, 16…）在二进制中只有一个 1：10000…0。

如果 n 恰好只有一个 1，那 n-1 借位后把该位变 0、右侧全变 1，AND 恰好得 0。

n = 0b 0001 0000 (=16, 只有一个1) n-1 = 0b 0000 1111 n & (n-1) = 0b 0000 0000 → 结果为 0 ➜ 是 2 的幂 n = 0b 0001 0100 (=20, 有两个1) n-1 = 0b 0001 0011 n & (n-1) = 0b 0001 0000 → 结果非 0 ➜ 不是 2 的幂

这个技巧背后是二进制借位的规律——学完本模块你将完全掌握它的来龙去脉。

📋 前置知识

二进制表示 补码（Two's complement）

二进制表示： 十进制 13 = 1×2³ + 1×2² + 0×2¹ + 1×2⁰ = 0b1101。每一位叫 bit，最低位（bit 0）在最右侧。

补码（Two's complement）： n 位有符号整数中，-x 的表示 = ~x + 1（按位取反再加 1）。

x = 0b 0000 0101 (= 5) ~x = 0b 1111 1010 -x = ~x+1 = 0b 1111 1011 (= -5, 8位补码)

这是 x & (-x) = lowbit 等技巧的数学基础。

Python 特殊性： Python 整数是无限精度的（无溢出），~x = -x-1（总是负数）。模拟 32 位运算需要 & 0xFFFFFFFF 掩码截断。

💡 动机：位运算 O(1) 替代循环

🐢 朴素做法：判断 2 的幂

def is_power_of_two(n):
    if n <= 0: return False
    while n > 1:           # O(log n) 循环
        if n % 2 != 0:
            return False
        n //= 2
    return True

⚡ 位运算：O(1)

def is_power_of_two(n):
    return n > 0 and (n & (n - 1)) == 0
    # 一条表达式，CPU 一两条指令

位运算之所以重要，不仅仅因为快：

  • 状态压缩： 用一个整数的每一位表示一个布尔状态，n 个状态只需一个 int，枚举所有子集 O(2ⁿ)。
  • 原子操作： 某些变换用位运算一行搞定（检测单个位、消去最低 1），代码简洁且语义清晰。
  • 数学性质： 异或的交换律、结合律使某类"找落单数"问题降到 O(1) 空间。

技巧一：x & (x-1) —— 清除最低位的 1

1
设 x = A1 0…0（最低位的 1 在某位，其右侧全 0）。
2
x-1 = A0 1…1（借位：最低 1 变 0，右侧全变 1）。
3
x & (x-1)：最低 1 位及右侧完全互补，AND 为 0；高位 A 不变。结论：消去最低位的 1。

技巧二：x & (-x) —— 取最低位的 1（lowbit）

1
-x = ~x + 1。设 x = A 1 0…0。
2
~x = Ā 0 1…1，加 1 后 -x = Ā 1 0…0（进位链）。
3
x & (-x)：高位 A & Ā = 0，最低 1 位 1&1=1，右侧 0&0=0。只保留最低位的 1。

技巧三：子集枚举 sub = (sub-1) & mask

1
sub - 1 借位：最低 1 变 0、右侧全变 1（可能出现 mask 中没有的位）。
2
再 & mask 过滤掉 mask 以外的位，得到下一个更小的子集。
3
循环直到 sub = 0，总枚举量 = 2^|mask|。对所有 mask 总计 O(3ⁿ)。

Brian Kernighan 算法：统计 1 的个数

核心：每次 n &= n-1 消去一个 1，循环次数 = popcount(n)。

n = 0b 1011 0100 (popcount = 4) 第1轮: n & (n-1) n = 1011 0100 n-1 = 1011 0011 & = 1011 0000 ← 消去了 bit2 的 1 count=1 第2轮: n & (n-1) n = 1011 0000 n-1 = 1010 1111 & = 1010 0000 ← 消去了 bit4 的 1 count=2 第3轮: n & (n-1) n = 1010 0000 n-1 = 1001 1111 & = 1000 0000 ← 消去了 bit5 的 1 count=3 第4轮: n & (n-1) n = 1000 0000 n-1 = 0111 1111 & = 0000 0000 ← 消去了 bit7 的 1 count=4 n=0 → 停止，共 4 轮 = 4 个 1 ✓
🤔 位掩码与集合映射：mask = 0b1011 对应哪个子集？枚举它的所有子集有几个？
展开答案

mask = 0b1011 = 11，第 0、1、3 位为 1 → 对应集合 {e₀, e₁, e₃}。

|mask| = 3（3 个 1），子集数 = 2³ = 8 个（含空集）。

枚举过程：1011 → 1010 → 1001 → 1000 → 0011 → 0010 → 0001 → 0000

推广： 对 n 位所有 mask 的子集总和 = Σ C(n,k)·2ᵏ = (1+2)ⁿ = 3ⁿ（二项式定理）。

🔷 位运算代数性质

性质公式意义
自反性a ^ a = 0任何数与自身异或为 0
零元a ^ 0 = a异或 0 不变
交换律a ^ b = b ^ a顺序无关
结合律(a ^ b) ^ c = a ^ (b ^ c)可任意分组
AND 单调性a & b ≤ min(a, b)AND 只能使值变小或不变
OR 单调性a | b ≥ max(a, b)OR 只能使值变大或不变
关键推论： 对一组数全部异或，两两出现的数消去，只留下落单的数。
a^b^b^c^a = (a^a)^(b^b)^c = 0^0^c = c（交换律 + 结合律 + 自反性）。

位运算不会溢出？ 严格说 AND / OR / XOR 不产生进位，结果位数不超过操作数位数。但左移可能溢出（在 C/Java 的固定位宽下），Python 无此问题。

🟢 典型例题

LC 191 位1的个数（Number of 1 Bits）

题意： 给定无符号整数 n，返回二进制中 1 的个数。

思路： Brian Kernighan 法——每次 n &= n-1 消去最低 1，计数循环次数。

def hammingWeight(n: int) -> int:
    count = 0
    while n:
        n &= n - 1
        count += 1
    return count
# n=11 (0b1011) → 3轮 → 返回 3
LC 136 只出现一次的数字（Single Number）

题意： 数组中每个元素出现两次，只有一个出现一次，找出它。要求 O(1) 空间。

思路： 全部异或，成对的抵消，剩下单独的。

def singleNumber(nums: list[int]) -> int:
    r = 0
    for x in nums:
        r ^= x
    return r
# [4,1,2,1,2] → 4^1^2^1^2 = 4
LC 260 只出现一次的数字 III（两个落单数）

题意： 数组中恰好两个数各出现一次，其余出现两次。找出这两个数。

1
全部异或得 xor = a ^ b（a, b 是两个目标数），xor ≠ 0。
2
取 diff = xor & (-xor)（lowbit），是 a, b 某一位不同的标记。
3
按 diff 将数组分两组，各组内部异或即得 a 和 b。
def singleNumber(nums: list[int]) -> list[int]:
    xor = 0
    for x in nums: xor ^= x
    diff = xor & (-xor)           # 分组依据
    a = b = 0
    for x in nums:
        if x & diff: a ^= x
        else:        b ^= x
    return [a, b]
LC 78 子集（位掩码法）

题意： 给定无重复整数数组，返回所有子集。

思路： n 个元素共 2ⁿ 个子集，用 0 到 2ⁿ-1 的整数作掩码，第 i 位为 1 表示选第 i 个元素。

def subsets(nums: list[int]) -> list[list[int]]:
    n = len(nums)
    result = []
    for mask in range(1 << n):
        subset = [nums[i] for i in range(n) if (mask >> i) & 1]
        result.append(subset)
    return result
# nums=[1,2,3] → [[], [1], [2], [1,2], [3], [1,3], [2,3], [1,2,3]]

🤔 如果每个元素出现三次只有一个出现一次（LC 137），还能用异或吗？
展开答案

不能直接用异或（三个相同数异或 = 原数而非 0）。需按位统计：每一位上 1 出现的次数 mod 3，余数就是目标数在该位的值。或者用两个计数变量模拟"三进制计数器"。

② 有符号右移 vs 无符号右移

语言>>无符号右移
C/C++实现定义（通常算术移位）用 (unsigned)x >> n

负数右移在不同语言行为不同，跨语言移植时务必注意。

位并行（Bitboard）： 用 64 位整数表示 8×8 棋盘，每位代表一个格子。一次 AND/OR/XOR 即可并行处理 64 个格子的状态（国际象棋引擎 Stockfish 的核心技术）。

GCC/Clang 内建函数（C/C++）：

函数功能等价

__builtin_ctz(x)末尾零个数最低位 1 的位置

这些通常编译为单条 CPU 指令（POPCNT, LZCNT, TZCNT），比手写循环快得多。

Gosper's Hack（枚举所有 popcount=k 的 mask）：

def gosper_hack(n, k):
    """枚举 n 位中恰好选 k 个的所有 mask"""
    mask = (1 << k) - 1          # 初始：最低 k 位全 1
    while mask < (1 << n):
        yield mask
        c = mask & -mask         # lowbit
        r = mask + c             # 进位
        mask = (((r ^ mask) >> 2) // c) | r

🔷 工业应用

权限系统位掩码： 用每一位表示一种权限，一个整数即可编码多个权限组合。

READ    = 1 << 0   # 0b0001
WRITE   = 1 << 1   # 0b0010
EXECUTE = 1 << 2   # 0b0100
ADMIN   = 1 << 3   # 0b1000

user_perm = READ | WRITE           # 授予读写
has_read  = bool(user_perm & READ) # 检查读权限 → True
user_perm &= ~WRITE                # 撤销写权限
# Linux 文件权限 chmod 755 = rwxr-xr-x 就是这个原理

布隆过滤器（Bloom Filter）： 用 m 位的位数组 + k 个哈希函数实现概率型集合查询。插入时设置 k 位为 1，查询时检查 k 位是否全为 1。可能误判（false positive），绝不漏判（no false negative）。广泛用于缓存穿透防护、垃圾邮件过滤。

SIMD（Single Instruction Multiple Data）： 现代 CPU 的向量指令集（SSE/AVX/NEON）在 128/256/512 位宽的寄存器上并行执行位运算，是图像处理、机器学习推理、高性能计算的基础——本质上是"超宽位运算"。
