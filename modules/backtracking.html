🔵 核心定义

回溯通用框架：

def backtrack(path, choices):
    # 终止条件：达到目标
    if 满足结束条件:
        result.append(path[:])   # 注意：要拷贝path，不能直接引用
        return

    for choice in choices:
        # 剪枝：提前排除无效分支
        if 不合法: continue

        # 做选择
        path.append(choice)
        mark_used(choice)

        # 递归进入下一层
        backtrack(path, new_choices)

        # 撤销选择（回溯）
        path.pop()
        unmark_used(choice)

三类经典问题的starter index控制：

  • 子集/组合： 从 start 开始，每次传 start+1（不重复使用）
  • 排列： 用 used[] 数组标记已用元素，每次从0开始遍历
  • 含重复元素： 先排序，同一层跳过 nums[i]==nums[i-1] 且未使用的

  同层中，若 nums[i]==nums[i-1] 且 used[i-1]==False
  则跳过（因为上一个相同元素已经生成过）

🟢 典型例题详解

例题1：全排列（LeetCode 46）

题意： 给不含重复数字的数组，返回所有全排列。

def permute(nums):
    result = []
    used = [False] * len(nums)

    def backtrack(path):
        if len(path) == len(nums):
            result.append(path[:])   # 必须拷贝！
            return
        for i in range(len(nums)):
            if used[i]: continue     # 跳过已使用元素
            used[i] = True
            path.append(nums[i])     # 做选择
            backtrack(path)
            path.pop()               # 撤销选择
            used[i] = False          # 恢复标记

    backtrack([])
    return result

时间： O(n × n!)；空间： O(n)

例题2：组合总和（LeetCode 39）— 可重复使用

题意： 从candidates中找和为target的所有组合，数字可重复使用。

def combinationSum(candidates, target):
    result = []
    candidates.sort()                # 排序方便剪枝

    def backtrack(start, path, remaining):
        if remaining == 0:
            result.append(path[:])
            return
        for i in range(start, len(candidates)):
            if candidates[i] > remaining:
                break                # 剪枝：后面的更大，不可能凑成
            path.append(candidates[i])
            backtrack(i, path, remaining - candidates[i])  # i不是i+1，可重复
            path.pop()

    backtrack(0, [], target)
    return result

时间： O(N^(T/M))，T为target，M为最小元素；空间： O(T/M)

例题3：N皇后（LeetCode 51）

题意： 在N×N棋盘放N个皇后，使之互不攻击。

def solveNQueens(n):
    result = []
    cols  = set()   # 已占用的列
    diag1 = set()   # 主对角线 row-col
    diag2 = set()   # 副对角线 row+col

    board = [['.']*n for _ in range(n)]

    def backtrack(row):
        if row == n:
            result.append([''.join(r) for r in board])
            return
        for col in range(n):
            if col in cols or (row-col) in diag1 or (row+col) in diag2:
                continue             # 该位置被攻击
            # 放置皇后
            board[row][col] = 'Q'
            cols.add(col); diag1.add(row-col); diag2.add(row+col)
            backtrack(row + 1)
            # 撤销
            board[row][col] = '.'
            cols.discard(col); diag1.discard(row-col); diag2.discard(row+col)

    backtrack(0)
    return result

时间： O(n!)；空间： O(n)

🔴 易错点

① 遗漏状态恢复导致污染后续分支

❌ 错误（忘记pop）

path.append(nums[i])
backtrack(path)
# 忘记 path.pop()
# 后续分支看到污染的path

✅ 正确（做-递归-撤）

path.append(nums[i])    # 做选择
backtrack(path)         # 递归
path.pop()              # 撤销（必须！）

② 结果收集时直接引用path而非拷贝

❌ 错误（引用同一对象）

result.append(path)
# path后续会被修改
# 所有result中的path都指向同一个列表

✅ 正确（深拷贝）

result.append(path[:])  # 切片拷贝
# 或 result.append(list(path))
# 或 result.append(path.copy())

🔷 竞赛拓展

Dancing Links（DLX）： 精确覆盖问题高效解法，用双向链表实现"删除-恢复"的O(1)回溯。适合数独、精确覆盖问题。

迭代加深IDDFS： 深度限制从1开始逐渐增大的DFS，兼顾BFS找最优和DFS低空间的优点。

启发式剪枝： 预计算"最优情况"，若当前剩余量在最优情况下也无法达标则剪枝。例如组合总和剪枝：if remaining < min_candidate: break
